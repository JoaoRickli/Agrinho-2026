# 🌱 AgroForte e Sustentável

Um site desenvolvido em **HTML, CSS e JavaScript** com o objetivo de promover a conscientização sobre a importância do agronegócio sustentável, destacando a relação entre produtividade agrícola, tecnologia e preservação ambiental.

## 📖 Sobre o Projeto

O AgroForte e Sustentável busca apresentar informações sobre práticas agrícolas responsáveis e os benefícios da sustentabilidade no campo. O site possui uma interface moderna, responsiva e intuitiva, sendo ideal para projetos escolares, apresentações acadêmicas e eventos relacionados ao agronegócio.

## ✨ Funcionalidades

- Página inicial com destaque para o tema.
- Navegação suave entre seções.
- Área informativa sobre sustentabilidade agrícola.
- Cards explicativos sobre tecnologia, preservação e futuro.
- Seção de benefícios do agro sustentável.
- Contador animado de árvores preservadas.
- Formulário de contato funcional.
- Design responsivo para diferentes dispositivos.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
